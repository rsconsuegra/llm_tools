# Character Completeness Check

Evaluate the character profile against these rules. Report violations (must fix), warnings (should fix), and observations (worth noting).

## Required Sections

The profile must contain all of these sections with non-empty content:

1. **Appearance** — build, distinguishing features, how they carry themselves
2. **Personality** — core traits, temperament, quirks
3. **Backstory** — formative events that shaped the character
4. **Motivations** — what drives them (wants vs needs)
5. **Voice** — speech patterns, vocabulary, example dialogue
6. **Arc** — starting state, turning points, projected end state

Violation if any section is missing or contains only placeholder text (TBD, TODO, N/A, placeholder).

## Specificity Rules

- **Appearance**: "Tall and beautiful" is a violation. Must include at least one concrete sensory detail (scar, gait, clothing texture, posture).
- **Personality**: Must list at least 3 distinct traits. At least one pair must create internal contradiction (e.g., generous but vindictive, brave but avoidant).
- **Backstory**: Must contain events, not descriptions. "She had a hard childhood" is a warning — "Her mother sold her to the guild at age nine for three silver coins" passes.
- **Motivations**: Must have both an external want and an internal need, and they must conflict. "Wants to find her brother but needs to forgive herself for leaving him" passes. "Wants to be happy" is a violation.
- **Voice**: Must include at least 2 lines of example dialogue. Without dialogue samples, the voice section is unverifiable.
- **Arc**: Must include a starting state and a projected end state. A turning point is required for protagonist/antagonist roles.

## Consistency Rules

- Personality traits must not contradict backstory events without explanation.
- Motivations must be traceable to backstory events.
- Voice must be consistent with age, background, and social class implied by other sections.
- Arc must be achievable within the implied scope of the story.

## Anti-Generic Rules

- Flag as a warning any trait that appears on this list without specific context: brave, kind, intelligent, strong, beautiful, mysterious, loyal, determined.
- These are not invalid — they are just too common to be distinctive without elaboration.
